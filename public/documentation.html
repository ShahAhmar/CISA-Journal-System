Getting Started

Journal Management

Creating a New Journal

1

Navigate to Journals

  1. Login as Admin
  2. Go to Admin Dashboard
  3. Click on "Journals" in the sidebar
  4. Click "Create New Journal" button
2

Fill Basic Information

Fill in the required fields:

  • Journal Name: Full name of the journal
  • Journal Initials: Short abbreviation
  • Slug: URL-friendly name (auto-generated)
  • Description: Brief description of the journal
  • ISSN: International Standard Serial Number
  • Print ISSN: Print version ISSN
  • Online ISSN: Online version ISSN
Note: Slug is automatically generated from journal name, but you can customize it.
3

Contact Information

Add contact details:

  • Primary Contact Name
  • Primary Contact Email
  • Contact Phone
  • Contact Address
  • Support Contact Name
  • Support Email
4

Content Sections

Configure journal content:

  • Aims & Scope: Journal's aims and scope
  • Focus & Scope: Detailed focus areas
  • Publication Frequency: How often journal publishes
  • Peer Review Process: Review process description
  • Peer Review Policy: Detailed review policy
  • Open Access Policy: Open access information
  • Copyright Notice: Copyright information
  • Privacy Statement: Privacy policy
5

Author Guidelines

Set up submission guidelines:

  • Submission Guidelines: General submission instructions
  • Author Guidelines: Detailed author instructions
  • Submission Requirements: IMPORTANT Requirements that appear in submission form
  • Submission Checklist: IMPORTANT Checklist items for authors
  • Competing Interest Statement: Conflict of interest policy
  • Copyright Agreement: Copyright terms
Important: Submission Requirements and Checklist are displayed to authors during submission. Make sure to fill these fields properly.
6

Editorial Team

Add editorial team information:

  • Editor-in-Chief
  • Managing Editor
  • Section Editors
  • Editorial Board
  • Editorial Team
7

Review Settings

  • Review Rounds: Number of review rounds (1-5)
  • Review Method: Double-blind, Single-blind, or Open
  • Requires Review: Whether review is mandatory
8

Website Settings

  • Logo: Upload journal logo
  • Cover Image: Journal cover image
  • Favicon: Browser icon
  • Theme: Color scheme
  • Homepage Content: Custom homepage content
9

Save Journal

  1. Review all information
  2. Click "Create Journal" button
  3. Journal will be created and you'll be redirected to journal list
Success! Your journal has been created. Now you can add sections and assign editors.

Section Management

Journal sections allow you to categorize submissions (e.g., Research Article, Review Article, Case Study, etc.)

1

Access Sections

  1. Go to Admin Dashboard → Journals
  2. Select a journal
  3. Click "Sections" tab or navigate to journal sections
2

Create New Section

  1. Click "Create Section" button
  2. Fill in the form:
    • Title: Section name (e.g., "Research Article")
    • Description: Brief description of the section
    • Section Editor: Assign a section editor (optional)
    • Word Limit Min: Minimum word count
    • Word Limit Max: Maximum word count
    • Review Type: Type of review for this section
    • Order: Display order (lower numbers appear first)
    • Active: Check to make section active
  3. Click "Create Section"
3

Edit Section

  1. Go to Sections list
  2. Click "Edit" on the section you want to modify
  3. Make changes
  4. Click "Update Section"
4

Deactivate Section

  1. Edit the section
  2. Uncheck "Active" checkbox
  3. Save changes
  4. Section will no longer appear in submission form
Note: Deactivating a section doesn't delete it. You can reactivate it later.

Common Section Types

Section Name Description Word Limit
Research Article Original research work 3000-8000
Review Article Literature review papers 5000-10000
Case Study Case study reports 2000-5000
Short Communication Brief research communications 1000-3000

Submission Process

Author Submission Guide

1

Login as Author

  1. Go to login page
  2. Enter your email and password
  3. If you don't have an account, click "Register" to create one
2

Select Journal

  1. After login, go to "PUBLISH WITH US" or homepage
  2. Browse available journals
  3. Click "Select" on the journal you want to submit to
3

Start Submission

  1. Click "Submit Article" or navigate to submission page
  2. You'll see a multi-step submission form
4

Step 1: Start Submission

  • Select Section: Choose appropriate section (Research Article, Review, etc.)
  • Submission Requirements: Review the requirements displayed
  • Submission Checklist: Review the checklist items
  • Confirm Requirements: Check the checkbox to confirm
  • Privacy Statement: Read and agree to privacy statement
  • Click "Next: Upload Files"
Important: Make sure you've read and understood all requirements before proceeding.
5

Step 2: Upload Files

  • Manuscript: Upload your main manuscript file (DOC or DOCX, max 10MB)
  • Title Page: Optional - separate title page
  • Cover Letter: Optional - cover letter to editor
  • Figures: Optional - separate figure files (PNG, JPG, JPEG)
  • Tables: Optional - separate table files (DOC, DOCX, XLS, XLSX)
  • Supplementary Files: Optional - additional supporting files
  • Click "Next: Enter Metadata"
6

Step 3: Enter Metadata

  • Title: Article title (required, max 500 characters)
  • Abstract: Article abstract (required)
  • Keywords: 3-5 keywords separated by commas
  • Supporting Agencies: Funding information (optional)
  • Click "Next: Add Authors"
7

Step 4: Add Authors

  • Add Author: Click to add author information
  • For each author, provide:
    • First Name (required)
    • Last Name (required)
    • Email (required)
    • Country (optional)
    • Affiliation (optional)
    • ORCID ID (optional)
  • First author is automatically set as corresponding author
  • You can add multiple authors
  • Click "Next: Review & Submit"
8

Step 5: Review & Submit

  1. Review all information:
    • Title
    • Abstract
    • Authors
    • Files
  2. Make any necessary corrections
  3. Click "Finalize Submission"
  4. You'll be redirected to success page
Congratulations! Your submission has been received. You'll receive email notifications about status updates.

Withdrawing a Submission

1

Withdrawal Process

  1. Go to "My Submissions"
  2. Open the submission you want to withdraw
  3. Scroll to "Withdraw Submission" section
  4. Click "Withdraw Submission" button
  5. Enter reason for withdrawal (required)

The editor dashboard provides an overview of all submissions for your journal(s) with statistics and quick actions.

1

Accessing Editor Dashboard

  1. Login with editor credentials
  2. You'll be redirected to editor dashboard
  3. If you have access to multiple journals, select a journal first
2

Dashboard Statistics

The dashboard shows statistics cards:

Viewing Submissions

1

Access Submission

  1. Go to "Recent Submissions" table or "All Submissions"

  3. Submission detail page will open
2

Submission Details

You'll see:

  • Title and Abstract

  • Uploaded Files (downloadable)

Editor Actions

1

Assign Editor

  1. Open submission detail page
  2. In right sidebar, find "Assign Editor" section
  3. Select an editor from dropdown
  4. Click "Assign"
2

Assign Reviewer

  1. In submission detail page, find "Assign Reviewer" section
  2. Select a reviewer from dropdown
  3. Set due date (must be future date)
  4. Click "Assign Reviewer"
  5. Reviewer will receive email notification
3

Accept Submission

  1. Click "Accept Submission" button
  2. Modal will open
  3. Add optional notes
  4. Click "Accept"
  5. Status will change to "Accepted" and move to copyediting stage
  6. Author will receive email notification
4

Reject Submission

  1. Click "Reject Submission" button
  2. Enter rejection reason (required)
  3. Click "Reject"
  4. Status will change to "Rejected"
  5. Author will receive email notification
5

Desk Reject

  1. For submissions that don't meet basic criteria
  2. Click "Desk Reject" button
  3. Enter reason (required)
  4. Click "Desk Reject"
  5. Rejected without sending to reviewers
6

Request Revision

  1. Click "Request Revision" button
  2. Select revision type:
    • Minor Revision
    • Major Revision
  3. Enter comments for author (required)
  4. Click "Request Revision"
  5. Status will change to "Revision Requested"
  6. Author will receive email notification
7

Publish Article

  1. For accepted submissions ready for publication
  2. Click "Publish Article" button
  3. Select issue (optional)
  4. Enter page numbers (optional)
  5. Click "Publish"
  6. Status will change to "Published"
8

Contact Author

  1. Click "Contact Author" button
  2. Enter email subject (required)
  3. Enter message (required)
  4. Click "Send Email"
  5. Email will be sent to corresponding author

Review Process

Reviewer Workflow

1

Receiving Review Invitation

  1. Reviewer receives email invitation
  2. Email contains link to review
  3. Reviewer can accept or decline
2

Accepting Review

  1. Login to system
  2. Go to Reviewer Dashboard
  3. Find pending review invitation
  4. Click "Accept" or "Decline"
  5. If accepting, set due date
3

Conducting Review

  1. Access review page
  2. Download and read manuscript
  3. Fill review form:
    • Recommendation (Accept, Minor Revision, Major Revision, Reject)
    • Comments for Editor
    • Comments for Author
    • Confidential comments (if any)
  4. Submit review

Review Recommendations

Recommendation Description
Accept Article is suitable for publication as is
Minor Revision Article needs minor corrections
Major Revision Article needs significant revisions
Reject Article is not suitable for publication

Troubleshooting

Common Issues

1

403 Forbidden Errors

Problem: Getting 403 errors when accessing pages

Solution:

  1. Check user role and permissions
  2. Ensure user is assigned to journal (for editors)
  3. Clear cache:
    4. php artisan route:clear php artisan config:clear php artisan view:clear php artisan cache:clear
2

Requirements/Checklist Not Showing

Problem: Submission requirements/checklist empty in form

Solution:

  1. Go to Admin Panel → Journals → Edit Journal
  2. Find "Author Guidelines" section
  3. Fill "Submission Requirements" field
  4. Fill "Submission Checklist" field
  5. Save journal
3

File Upload Issues

Problem: Files not uploading

Solution:

  1. Check file size (max 10MB for manuscript)
  2. Check file format (DOC/DOCX for manuscript)
  3. Check storage permissions
  4. Check PHP upload limits in php.ini
4

Email Not Sending

Problem: Email notifications not being sent

Solution:

  1. Go to Admin Panel → System Settings
  2. Configure SMTP settings
  3. Test email configuration
  4. Check email logs
